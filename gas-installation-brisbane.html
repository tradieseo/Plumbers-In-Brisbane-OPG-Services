Frequently Asked Questions

Can you install customer-supplied appliances?
Yes. Customer-supplied cooktops, ovens, heaters and hot water units can be installed as long as the appliance is suitable for the property and meets Australian requirements.
Can you install an appliance where there isn’t an existing connection point?
Yes. If there’s no current connection available, a new line or connection point can be installed so the appliance can be fitted correctly.
Do you install both Natural Gas and LPG appliances?
Yes. Installations can be completed for both Natural Gas and LPG appliances, including correct connection requirements based on the gas type used at your property.
Do you handle installations for renovations and kitchen upgrades?
Yes. Installation work can be scheduled to suit renovation timelines, including new appliances, relocations, and new connection points for updated layouts.
How much does gas installation in Brisbane cost?
Cost depends on the type of appliance, whether the connection point already exists, and access to the install location. A free quote can be provided once the job details are confirmed.
